                                                                                                                                                                    Matthew Lein Mod kaushi • 7 years ago • edited

                                                                                                                                                                    CSS transitions work in all modern browsers, your text editor will not affect that.

                                                                                                                                                                    Your problem could be many things, so I will give you the tips to troubleshoot and learn anything new:
                                                                                                                                                                    -Find a working example online
                                                                                                                                                                    -Does it work? Yes.
                                                                                                                                                                    -Copy it to your page
                                                                                                                                                                    -Does it work?

                                                                                                                                                                    -If not, try a new page.
                                                                                                                                                                    -Paste it into a new codepen/jsfiddle
                                                                                                                                                                    -Does it work?
                                                                                                                                                                    -If not you missed something from the working online version

                                                                                                                                                                    -If Yes, edit their code to use your page element
                                                                                                                                                                    -Test again

                                                                                                                                                                    -If it works, compare your original attempt with the now working version and see what the difference is.
                                                                                                                                                                    -If not, go back to the working version and see how many small changes you can make until it breaks.
